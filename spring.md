# Spring Framework


Spring is Dependency Injection Framework.
Spring Framework provides many sub-frameworks AOP,ORM,JDBC,MVC etc.

What is dependency.?

* WebLayer

    * JSP,Servlet,MVC,WEB, VIEW

* Business Layer
 

* Data Layer

    * Hibernate
    * JDBC
    * IBATIS
